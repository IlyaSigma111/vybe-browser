// features

Built for speed.
Crafted for control.

>_

Vim Keybindings

Navigate the web without touching your mouse. j/k to scroll, gg/G to jump, o to open URLs.

[]

Command Mode

Press : to enter command mode. Open tabs, reload pages, zoom — all from the keyboard.

///

Instant Search

Press / to search. Type your query and hit enter. No need to reach for the mouse.

[ ]

Tab Management

t for new tab, q to close, J/K to switch. Number keys 1-9 for direct access.

//

Trapezoid Tabs

Classic Chrome-style trapezoid tabs. Because some designs never go out of style.

{ }

Tauri Powered

Built on Tauri for minimal resource usage. 48MB install, starts in milliseconds.

JNext tab
KPrev tab

:Command mode
/Search
?Help
